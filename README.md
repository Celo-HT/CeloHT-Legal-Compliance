## Overview
CeloHT is a decentralized Web3 platform built on **three fundamental pillars**: **Education**, **Agents**, and **Reforestation**, leveraging **Celo cUSD** and **Valora** for transactions and community support. **CeloHT is not a cryptocurrency or token**; it is a platform supporting the Celo ecosystem.



## FAQ (Frequently Asked Questions)

**Q1: Is CeloHT a cryptocurrency?**  
A: No, CeloHT is a platform that supports Celo cUSD and Valora but is **not a token or crypto**.  

**Q2: Can CeloHT provide financial advice?**  
A: No, all financial decisions are the responsibility of the user.  

**Q3: How is user data protected?**  
A: CeloHT adheres to GDPR, CCPA, and other relevant international data protection standards.  

**Q4: Who governs CeloHT?**  
A: Johnny Dubic as Founder, with a **community-driven governance committee** overseeing operations.
